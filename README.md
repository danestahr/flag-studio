# Flag Studio

Internal design tool for creating golf tournament flag and hole sign packages — multi-step wizards for selecting styles, assigning colors, uploading logos, building variations, and exporting print-ready files. Backed by Supabase (auth, Postgres, Storage, edge functions).

See `CLAUDE.md` for the full page map, state model, and Supabase schema.

## Running locally

```bash
npm install
npm run dev
```
